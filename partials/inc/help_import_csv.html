Cette fonction permet d'insérer dans la base des contacts stockés sous forme de feuille de calcul. Le format utilisé est le fichier csv (comma separated values). C'est un fichier texte ou chaque ligne représente un contact et chaque champ est séparé par une virgule. Les champs de type texte sont encadrés par le caractère ".

LibreOffice Calc par exemple permet de générer de tels fichiers.

Préparer son fichier csv

Votre fichier doit comporter les noms de colonnes dans la première ligne. Les colonnes utilisées par le sytème d'import correspondent aux noms de champs suivants : type, nom, prenom, email, tel, note, fonction, adresse, cp, ville, pays. Les colonnes de type id, idstr, tag, permettent de faire des liens entre individus et structures et entre contacts et catégories (tags).

Les champs uniques

Pour certains types de champ il ne peut y avoir qu'une seule colonne : il s'agit des champs nom, prenom, type, fonction, adresse, cp, ville, pays. S'il y en a plusieurs, seule la première colonne sera prise en compte. Le champ type indique si le contact est un individu (type=1) ou une structure (type=2), si c'est une structure, le champ prenom sera ignoré.

Les champs de type note peuvent apparaitre plusieurs fois, le contenu sera mis à la ligne dans la note du contact.

Les champs multiples

Les champs tel et email peuvent apparaitre plusieurs fois, vous aurez la possibilié de les nommer au moment de l'import ("Téléphone fixe", "Téléphone portable" par exemple).

Associations individu / structure

Vous avez la possibilité d'associer un (ou plusieurs) individu(s) à une structure. Il faut pour cela ajouter deux colonnes : id et idstr. Le champ id doit être renseigné avec une valeur de votre choix (un numéro par exemple) sur la ligne correspondant à la structure. Cette valeur doit être reportée sur la ligne correspondant à l'individu dans le champ idstr. Attention : la valeur de la colonne id doit être différente pour chaque structure.

Exemple :

ididstrtypenomprenom
12Radio France
11MartinNicolas

Tags

Les tags ou catégories peuvent être associés aux contacts en ajoutant des colonnes tag. S'il y a plusieurs tags dans chaque colonne, il suffit de les séparer par des virgules. On peut aussi préciser dans le nom de colonne un tag parent pour les tags de la colonne (tag|Nom du tag parent).

Exemple :

typenomprenomtag|Thématique
1MartinNicolasPhysique,Chimie,Éthique

Le contact sera associé aux tags suivants : Thématique>Physique, Thématique>Chimie et Thématique>Éthique.

Importer votre fichier

Vous devez commencer par choisir un tag qui sera associé à tous les contacts concernés par l'import. Il est préférable d'en choisir un qui n'est associé à aucun contact, cela vous permettra d'annuler facilement l'import en cas de problème. Vous pouvez en choisir plusieurs, mais seul le premier se verra attribuer les éventuels sous-tags. Si on reprend l'exemple ci dessus, en ayant choisi comme tags pour l'import "import n°1" et "radio", l'import créera les tags suivants : import n°1>Thématique>Physique, import n°1>Thématique>Chimie et import n°1>Thématique>Éthique

Un fois l'upload du fichier terminé, le logiciel vous indique les champs qu'il a trouvé. Pour certains, vous pouvez les renommer si besoin (tel, email). En dessous, vous pouvez parcourir les 100 premiers contacts de votre import pour vérifier que les informations sont bien prises en compte.

Vous pouvez alors soit modifier votre fichier et le charger à nouveau, ou bien, si tout est bon, valider l'import.
